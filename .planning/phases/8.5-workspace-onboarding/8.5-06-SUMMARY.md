## Lazy Daemon Pattern

```
bridge init --workspace /path/to/repo
  |
  v
GET /health (2s timeout)
  |--- ok ----> POST /workspaces/init
  |
  `--- fail --> spawn bridge-daemon (detached)
                  |
                  v
                poll GET /health every 500ms (15s timeout)
                  |--- ok ----> POST /workspaces/init
                  `--- timeout -> error + exit 1
```
